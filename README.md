# Conversor-word-pdf
 Conversor de arquivo Word para PDF

## Linguagem
Python versão 3.10.0

## IDE
VsCode com Python

## Bibliotecas utilizadas
docx2pdf==0.1.8
PySimpleGUI==4.56.0

## Escopo

Tela feita no PySimpleGUI onde podemos extrair o arquivo WORD e transformar em PDF.
O Status é mostrado após a conversão.
